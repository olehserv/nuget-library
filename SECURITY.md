# Security Policy

## Supported Versions

This project is a test assignment and does not have long-term support versions.

| Version | Supported |
|---------|-----------|
| Latest  | ✅       |
| Older   | ❌       |

Only the latest version of the code in the `main` branch is considered supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Instead, report it responsibly by contacting the repository owner directly:

- Create a **private GitHub Security Advisory**, or
- Contact the maintainer via GitHub profile messaging

Please include as much detail as possible:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigation (if available)

You can expect an initial response within **7 days**.

---

## Scope

This security policy applies only to:
- Source code contained in this repository
- Dependencies explicitly defined in the project

Infrastructure, CI/CD runners, and third-party services are out of scope.
